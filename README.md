# 🤿 World Diving Spots

세계 최고의 스쿠버 다이빙 포인트를 한눈에 탐색할 수 있는 인터랙티브 지도 가이드.

**🌊 라이브 데모 → [redhands.github.io/diving-spots](https://redhands.github.io/diving-spots/)**

---

## 주요 기능

- **인터랙티브 세계 지도** — 위성 지도 위에 50개 다이빙 스팟 마커 표시, 무한 횡스크롤 지원
- **월별 시즌 필터** — 월을 선택하면 해당 달에 가기 좋은 스팟만 하이라이트, 나머지는 숨김
- **국가별 그룹 목록** — 우측 패널에 27개국을 국가명 기준으로 분류해서 표시
- **상세 정보 팝업** — 스팟 클릭 시 수심·시야·수온·베스트 시즌·설명·주요 해양 생물 표시
- **모바일 대응** — FAB 버튼 + 바텀 시트로 모바일 화면에서도 전체 기능 사용 가능
- **동적 통계** — 전체 스팟 수·추천 스팟 수·생물 종류·국가 수를 JSON 데이터 기반으로 자동 계산

## 스팟 데이터 추가 방법

`spots.json`에 아래 형식으로 객체를 추가하면 자동으로 지도와 목록에 반영됩니다.

```json
{
  "id": 51,
  "name": "스팟 이름",
  "nameEn": "Spot Name",
  "country": "국가명",
  "lat": 0.0,
  "lng": 0.0,
  "color": "#3b82f6",
  "icon": "🐠",
  "bestMonths": [3, 4, 5, 6],
  "difficulty": "초급~중급",
  "depth": "5~30m",
  "visibility": "15~25m",
  "waterTemp": "24~28°C",
  "creatures": ["생물1", "생물2", "생물3"],
  "desc": "스팟 소개 텍스트"
}
```

| 필드 | 설명 |
|------|------|
| `id` | 고유 번호 (기존 최대값 + 1) |
| `lat` / `lng` | 위도 / 경도 |
| `color` | 마커 색상 (hex) |
| `bestMonths` | 추천 월 배열 (1–12) |
| `difficulty` | 난이도 (`초급` / `중급` / `고급` 등) |
| `creatures` | 주요 해양 생물 목록 |

## 기술 스택

| 역할 | 기술 |
|------|------|
| 지도 엔진 | [Leaflet.js](https://leafletjs.com/) v1.9.4 |
| 지도 타일 | ESRI World Imagery (무료, API 키 불필요) |
| 폰트 | [Nanum Gothic](https://fonts.google.com/specimen/Nanum+Gothic) |
| 배포 | GitHub Pages |
| 빌드 도구 | 없음 (순수 HTML/CSS/JS) |

## 로컬 실행

```bash
# Python 내장 서버 사용
python3 -m http.server 3000 --directory diving-spots

# 브라우저에서 열기
open http://localhost:3000
```

> `spots.json`을 `fetch()`로 불러오기 때문에 파일을 직접 열면(`file://`) 동작하지 않습니다. 반드시 HTTP 서버를 통해 실행하세요.
